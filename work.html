Mon Approche de Conception

Recherche Utilisateur

Je commence par comprendre les besoins des utilisateurs à travers des interviews et l'analyse des données existantes.

Prototypage

Je crée des prototypes pour tester rapidement des idées avant le développement final.

Tests Utilisateurs

Je valide mes conceptions avec de vrais utilisateurs pour identifier les problèmes d'utilisabilité.

Implémentation

Je travaille avec les développeurs pour m'assurer que le design est bien respecté dans le produit final.

Mon Apprentissage

Dans le cadre du cours SEG3525 à l'Université d'Ottawa, j'apprends :

  • Les principes de base de la conception UX/UI
  • Les méthodes d'évaluation des interfaces
  • Le développement front-end avec HTML, CSS et JavaScript
  • L'utilisation de frameworks comme Bootstrap

Je complète cette formation par des recherches personnelles et la pratique sur des projets concrets.
